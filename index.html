🔩 Equipamentos Principais da Subestação

Uma subestação industrial típica é composta pelos seguintes equipamentos principais:
  • Transformador de força: converte a tensão de média para baixa tensão. Classe F ou H.
  • Chave seccionadora (CS): isolamento e manobra sem carga do circuito primário.
  • Para-raios: proteção contra sobretensões atmosféricas (13,8 kV → tipo ZnO).
  • Disjuntor de MT: proteção e manobra do primário do transformador.
  • Transformadores de corrente (TC) e tensão (TP): para instrumentação e proteção.
  • Quadro de MT (QDG/QGBT): distribuição em baixa tensão com barramento, DPS e medição.
  • Banco de capacitores: correção do fator de potência (se necessário).
  • Sistema de aterramento: malha de terra conforme NBR 7117.

Integração Claude AI + Revit via MCP

Conecte o Claude diretamente ao Revit usando o protocolo Model Context Protocol (MCP) e automatize projetos elétricos com inteligência artificial.

O que é o MCP?

O Model Context Protocol (MCP) é um protocolo aberto criado pela Anthropic que permite que modelos de linguagem como o Claude se conectem a ferramentas e sistemas externos de forma padronizada. Em vez de copiar e colar dados manualmente, o Claude consegue ler, criar e modificar elementos diretamente dentro do Revit em tempo real.

Com isso é possível, por exemplo, pedir ao Claude para "colocar uma luminária de 2x40W em cada cômodo do pavimento tipo conforme a NBR 5410" e ele executa isso automaticamente no modelo BIM.

📦 Repositório — PyRevit MCP Server

O servidor MCP para Revit é mantido como projeto open-source. Ele usa PyRevit como ponte entre o Claude e a API do Revit.

🚀 Passo a Passo — Instalação

1
Instalar o PyRevit
Baixe e instale o PyRevit no site oficial. Após a instalação, o Revit terá uma aba extra "pyRevit" na ribbon.
⬇️ Download PyRevit
2
Clonar o repositório MCP
Abra o terminal (CMD ou PowerShell) e execute:
git clone https://github.com/troopl/revit-pyrevit-mcp
3
Instalar dependências Node
Dentro da pasta clonada, execute:
cd revit-pyrevit-mcp
npm install
4
Iniciar o Revit com PyRevit
Abra o Revit normalmente. O PyRevit carrega automaticamente na inicialização. Confirme que a aba PyRevit aparece na ribbon.
5
Iniciar o servidor MCP
No terminal, dentro da pasta do projeto, rode:
npm start
O servidor ficará escutando na porta 3000 por padrão.
6
Configurar o Claude
No Claude.ai (plano Pro) ou no Claude Desktop, vá em Configurações → Ferramentas MCP e adicione o servidor local:
http://localhost:3000
7
Testar a conexão
No chat do Claude, tente: "Liste os elementos do modelo Revit atual". Se o servidor estiver rodando e o Revit aberto, o Claude responderá com os dados do modelo.

"Liste todas as luminárias do pavimento térreo e me diga se atendem a NBR 5410 para o cômodo correspondente."
"Coloque uma luminária de embutir 2x40W em cada sala do modelo, espaçadas conforme a NBR 5410."
"Calcule a carga total de iluminação do pavimento tipo e indique o disjuntor necessário."
"Renomeie todos os painéis elétricos do modelo seguindo o padrão QD-01, QD-02..."
"Gere um relatório em tabela com todos os circuitos elétricos, seção do cabo e disjuntores."
"Verifique se há luminária faltando em algum cômodo conforme a planta baixa atual."

⚠️ O servidor MCP roda localmente na sua máquina — nenhum dado do seu projeto é enviado para servidores externos além da API da Anthropic (que processa apenas o contexto textual do chat). Para projetos sigilosos, revise a política de privacidade da Anthropic antes de usar.

• A corrente flui apenas em circuitos fechados

Referência ANSI/IEEE 315 e IEC

SIMBOLOGIA

Contatos

Segundo ANSI/IEEE 315 e IEC 60617, os contatos são representados por pares de linhas com uma barra inclinada que pode ser conectadas a outra extremidade (NF) ou não (NA):

Contato NA (Normalmente Aberto)
Normally Open — fecha quando a bobina é energizada
Contato NF (Normalmente Fechado)
Normally Closed — abre quando a bobina é energizada

Bobinas

A bobina é representada por um círculo/fiação enrolada (IEEE C37.2) ou Retangulo (IEC 60617). Quando energizada, atua sobre todos os contatos com o mesmo identificador:

Bobina de Contator (M / KM)
Energiza todos os contatos com mesmo ID
Bobina (padrão IEC)
Energiza todos os contatos com mesmo ID

Botões Pulsadores

O símbolo do botão pulsador inclui a letra T (actuator) sobre o contato. A seta indica o acionamento manual:

Botão Pulsador NA — "S" (Start)
Fecha momentaneamente ao ser pressionado
Botão Pulsador NF — "P" (Parada/Stop)
Abre momentaneamente ao ser pressionado

Nomenclatura ANSI

Designações padrão ANSI para dispositivos em diagramas de comando elétrico:

52. Disjuntor de corrente alternada
27. Relé de subtensão
43. Seletor de modo
50/51. Relé de sobrecorrente
59. Relé de sobretensão
67. Relé de sobrecorrente direcional
87. Relé diferencial
Não possuem um especifico — Lâmpada sinalizadora (pilot lamp)
33.1 — Fim de curso (limit switch)

Chaves Seletoras

Chaves seletoras possuem um símbolo de alavanca e indicam a posição (0, 1, 2, etc.).

43. Seletor de modo
43.1 — Seletora 3 posições
43.2 — Seletora 4 posições

Nomenclatura IEC 81346

Designações padrão IEC para identificação de dispositivos em diagramas:

K / KA — Relés e Contatores Auxiliares
KM — Contator de Potência (Motor)
Q — Disjuntor / Seccionadora
F — Fusível / Relé Térmico
S — Botões e Chaves Seletoras
H — Sinalização (Lâmpadas/Sirenes)
M — Motores Elétricos
T — Transformadores
B — Sensores e Fins de Curso
X — Bornes e Conectores
